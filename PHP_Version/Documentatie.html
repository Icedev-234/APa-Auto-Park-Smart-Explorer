2.Descriere generala

2.1 Descrierea Produsului

Aplicatia Auto Park Smart Explorer, este developata pentru persoanele care vor sa caute informatii despre parcul auto dintr-un anumit judet din Romania, in functie de tipul de vehicul, categorie sau ani.

2.2 Functiile Produsului

Aplicatia web managerieaza informatiile despre parcul auto dintr-un judet ce face parte din Romania. Despre acest parc sunt informatii precum : anul in care s-a facut masuratoarea pentru parc, valoarea propriu zisa ce reprezinta numarul de masini aflate in parc in anul respectiv, vizualizarea datelor folosind Map, Table si Line. Exportarea charturilor in unul din formatele: CSV, WEBP, SVG si selectarea unui singur judet asupra caruia ii se vor aplica filtre.

Statistics/Index page : Pagina din care pot fi accesate celelalte pagini ale website-ului

  • login : redirectioneaza catre login page.
  • contact : redirectioneaza catre contact page.
  • about : redirectioneaza catre about page.
Login page :
  • log in : valideaza logarea utilizatorului si il redirectioneaza catre index page
Admin page :
  • login : redirectioneaza catre login page.

2.3 Clasele și caracteristicile utilizatorilor

  • User : user asociat cu un cont poate utiliza facilitatile website-ului.
  • Admin : are optiunea de a manageria aspecte diferite ale website-ului.

2.4 Mediul de operare

Pagina web poate fi utilizata pe orice device care are o conexiune la internet si un browser instalat.

2.5 Constrângeri de proiectare și implementare

Auto Park Smart Explorer va fi dezvoltat in Php/Python, si va folosi html, css si javascript pentru interfata.

4. System Features

4.1 Statistics/Index page

Oaspeții și utilizatorii pot naviga la toate celelalte pagini de pe această pagină. De asemenea, utilizatorii pot cauta diferite judete asupra carora se aplica filtre si are posibilitatea de a vizualiza rezultatul sub forma unui chart la alegere.

4.2 Login page

Utilizatorii aplicației se pot autentifica introducând informatiile corecte și, dupa validare, accesează alte funcționalități ale aplicației.

4.3 Admin page

Utilizatorul cu rol de admin al aplicației poate sa insereze, sa modifice sau sa stearga datele. De asemenea, acesta poate sa le caute dupa anumite filtre.

4.4 Security_and_policy

Utilizatorul deja logat in aplicației poate sa isi modifice parola cu care are loc actiunea de login.

4.5 About page

Utilizatori conectati sau nu pot observa informatiile despre pagina noastra

4.6 Contact page

Utilizatori conectati sau nu pot observa informatii despre dezvoltatorii aplicatiei

5. Cerinte Nonfunctionale

5.2 Safety Requirements

Pentru a se asigura că niciunul dintre utilizatorii Auto Park Smart Explorer nu pierde date în timpul utilizării aplicației, echipa de dezvoltatori lucrează în mod regulat la Auto Park Smart Explorer.

5.3 Cerinte de securitate

Auto Park Smart Explorer nu are cerințe de securitate și, prin urmare, orice tip de utilizator îl poate folosi fără privilegii suplimentare.
